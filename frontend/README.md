# MOVE FRONENT

This project is the frontend service for Move App.

##### The project structure is:

```bash
├── src
│   ├── components
│   ├── pages
│   ├── services
│   └── styles
└── package.json
```

## Quick start

- Navigate for this folder:

- Install dependences:

`yarn`

- Running it for dev environment:

`yarn start`
